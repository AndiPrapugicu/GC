# GC - Laboratory Submission

Acest repository contine materialele pentru laboratoarele de Computer Graphics (OpenGL ES 1.x) si exercitiile de introducere Android.

## Continut
- `solutions/` - fisierele de solutie pregatite pe laboratoare
- `android_projects_snapshot/` - snapshot de cod din proiectele Android Studio create
- PDF-urile de laborator folosite ca cerinte

## Laboratoare
1. Lab 1 - HelloWorld OpenGL ES
2. Lab 2 - BouncySquare (2D)
3. Lab 3 - BouncyCube (3D)
4. Intro Android Studio - TextView + Button + Toast

## Rulare
1. Deschide fiecare proiect in Android Studio.
2. Selecteaza un emulator/device.
3. Ruleaza configuratia `app`.

## Observatii
- In acest repo sunt incluse fisierele relevante de cod pentru evaluare.
- Build-urile locale au fost validate inainte de export.
