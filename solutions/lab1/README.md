# Laborator 1 - HelloWorld OpenGL ES 1.x

## Fisiere
- `HelloWorldActivity.java`
- `HelloWorldRenderer.java`

## Ce trebuie sa faci in Android Studio
1. Creezi proiect nou: `HelloWorld`, package `cg.helloworld`.
2. Minimum SDK: API 8 (Android 2.2), Empty Activity, fara layout XML.
3. Inlocuiesti clasele generate cu fisierele din acest folder.
4. Rulezi pe emulator.

## Verificare
- Ecran complet rosu (clear color setat pe rosu).
- Aplicatia porneste fara erori.

## Observatie
Pentru test rapid, modifica in renderer valorile din `glClearColor(r, g, b, a)`.
