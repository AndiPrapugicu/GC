# Lab 6 - Blending

This solution extends `BouncySquare` with three OpenGL ES 1.x demos:

- alpha blending between two colored squares
- texture blending with two textures and additive blending
- multitexturing using two texture units

Assets used:

- `hedly.png`
- `black_stone.avif`

Run the `BouncySquare` project in Android Studio or with Gradle after setting `JAVA_HOME` to the Android Studio JBR and `sdk.dir` in `local.properties`.
