# Introducere Android Studio - ex1 + ex2

## Fisiere
- `MainActivity.java`
- `activity_main.xml`

## Cum le folosesti
1. Creezi proiect nou `MyFirstApp` in Android Studio.
2. Copiezi `MainActivity.java` in pachetul aplicatiei.
3. Copiezi `activity_main.xml` in `res/layout`.
4. Rulezi pe emulator.

## Ce obtii
- Textul "Salut, Android!" apare in centru.
- Butonul "Schimba Culoarea" schimba succesiv culoarea textului.
- Se afiseaza mesaj `Toast` la fiecare apasare.
